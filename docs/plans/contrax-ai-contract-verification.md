## Context (배경)

### 현재 상태
- 계약서 검토는 법률 전문가가 수동으로 수행하며, 비용(건당 50~200만원)과 시간(3~7일) 부담이 큼
- 비전문가(스타트업 대표, 프리랜서 등)는 불공정 조항을 인지하지 못한 채 서명하는 경우 빈번
- 단일 AI 검토 시 편향(hallucination, 특정 관점 누락)이 발생할 수 있어 신뢰도 한계

### 현재 구현 상태 (v1.0)
- Next.js 16 + TypeScript + Tailwind CSS 기반 웹앱
- PDF 업로드 → pdf-parse 텍스트 추출 + CLOVA OCR (스캔 문서)
- Claude Sonnet 4.6 + GPT-5.4 + Gemini 2.5 Flash 3개 AI 병렬 분석
- SSE 기반 실시간 진행상황 스트리밍
- 교차검증 합의율 산출 (axis-kit xv 방식: 90%+ auto_approve, 70-89% human_review, <70% redefine)
- 좌우 패널 풀와이드 레이아웃 (좌: PDF원본/텍스트하이라이트, 우: 분석결과)
- 원문 하이라이트 + 이슈 양방향 연동 + 사이드패널 상세 + 키보드 탐색
- 심각도/유불리 필터링, 대시보드 시각화(도넛차트, 바차트), HTML 리포트 다운로드
- 로컬 히스토리 최대 5개 저장

### 왜 필요한가
- **비용 절감**: 전문가 검토 전 1차 스크리닝으로 검토 시간/비용 80% 절감
- **교차검증 신뢰도**: 단일 AI 대비 3개 AI 독립 분석 + 합의율 산출로 편향 최소화
- **접근성**: 법률 비전문가도 계약서 리스크를 쉽게 파악 + 수정 문구까지 제안받을 수 있음
- **갑을 관계 시각화**: 발주자/공급자 유불리를 명시적으로 보여줘 협상력 확보

### 관련 자료
- axis-kit /xv UX 교차검증 결과: `docs/verifications/2026-03-27-ContraX-계약서-교차검증-웹앱의.md`
- API 키 관리: `swk-cloud-manage/.secret/external/ai-services.md`

## Problem (문제 정의)

### 핵심 문제
**계약서 검토의 높은 비용과 시간 장벽을 AI 교차검증으로 해결하되, 단일 AI의 신뢰도 한계를 극복해야 한다.**

### MECE 분해

| # | 문제 영역 | 설명 | 영향도 |
|---|----------|------|--------|
| 1 | 문서 처리 | PDF/이미지/스캔 문서에서 정확한 텍스트 추출이 필요 | 높음 |
| 2 | AI 분석 정확도 | 단일 AI는 편향/누락이 있으므로 다중 AI 교차검증 필요 | 높음 |
| 3 | 합의 알고리즘 | AI 간 의견 차이를 정량화하고 신뢰할 수 있는 합의율 산출 | 높음 |
| 4 | 유불리 판단 | 계약 조항이 어느 당사자에게 유리/불리한지 명확히 식별 | 중간 |
| 5 | 수정 제안 | 문제 조항에 대해 실제 대체 가능한 수정 문구 제공 | 중간 |
| 6 | 사용자 경험 | 비전문가도 쉽게 사용할 수 있는 직관적 UI/UX | 중간 |
| 7 | 분석 결과 활용 | 결과를 리포트로 저장하고 이전 분석을 히스토리로 관리 | 낮음 |

### 제약 조건
- **기술적**: 3개 AI API 동시 호출 비용 (건당 약 $0.03~0.10), 응답 시간 30~60초
- **법적**: AI 분석은 법률 자문이 아님을 명시해야 함 (면책 고지 필수)
- **정확도**: 스캔 문서의 OCR 품질에 따라 분석 정확도가 달라짐
- **보안**: 계약서는 민감 문서이므로 서버 저장 최소화, 클라이언트 기반 처리 권장

## Solution (해결 방안)

### 선택한 방안
**Next.js 풀스택 + 3개 AI 병렬 교차검증 + axis-kit 합의 프로토콜**

### 대안 비교

| 기준 | A. 멀티AI 교차검증 | B. 단일 AI (Claude Only) | C. 전문 법률 AI SaaS |
|------|-------------------|------------------------|---------------------|
| 신뢰도 | 높음 (3AI 합의) | 중간 (편향 가능) | 높음 (학습 데이터) |
| 비용 | 건당 ~$0.05 | 건당 ~$0.02 | 월 $100~500 |
| 구현 난이도 | 중간 | 낮음 | 없음 (SaaS) |
| 커스터마이징 | 완전 자유 | 완전 자유 | 제한적 |
| 유불리 분석 | 가능 (교차검증) | 가능 (단일 관점) | 제공 여부 불확실 |
| 선택 | **채택** | 기각 (신뢰도 부족) | 기각 (비용 + 종속성) |

### 구현 범위

### 검증 기준
1. PDF 업로드 → 3개 AI 분석 완료 → 합의율 산출까지 60초 이내
2. 공통 지적사항 정확도: 실제 법률 검토와 70% 이상 일치
3. 원문 하이라이트 매칭률 50% 이상 (현재 약 60%)
4. 모든 UI 인터랙션이 200ms 이내 반응
5. 브라우저 콘솔에 에러 없음
